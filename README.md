# skisim
Ski simulation for investigating accident situations, suitable for experts and specialists.


Objects: Drag objects from menu to add to scene.
Move with mouse, fingers or with arrow keys <--> ^ v

Rotate: Right mouse button / ö ä keys.

Scale: Mouse wheel / + - keys.

Text/Vector: Enter or double-click.

Change Color (box/lines): C key.

Switch: Tab (also "no focus").

Focus: Click on an empty area to remove focus.

Input numbers: You may give text input or number input for vector object. 

Result of simulation:
You may also save a PNG screenshot of the resulting scene using the "download" button or with pressing "e".
You may share the PNG screenshot through your OS sharing possibilities using the "share" button. 
You may save the resulting scenery in a JSON document using the "save" button or with pressing "s". 
You may restore a scenery in a JSON document using the "import" button or with pressing "i".
